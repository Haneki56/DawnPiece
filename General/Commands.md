# Commands

This page lists all of the commands that you will be able to use on the server as a player. Some commands however, may be blocked behind certain donation ranks and other factors. Please see the list of commands by mod and plugin.

### DawnPiece

The following chart lists all of the DawnPiece Commands that you are able to use in game.

| Command                  | Description                                  | Access        |
|-----------               |-----------                                   |-----------    |
| /dp ranks                | Shows clickable options to pick a life path  | Default       |
| /dp ranks set (rank)     | Sets your own life path                      | Default       |
| /dp ranks clear          | Clears your life path                        | Default       |
| /dp bounty               | Shows the bounty help menu                   | Default       |
| /dp bounty check         | Checks your own bounty                       | Default       |
| /dp bounty check (player)| Checks another playerâ€™s bounty               | Default       |
| /dp bounty list (page)   | Shows the leaderboard for all server bounties| Default       |
| /dp bank                 | Opens your bank GUI                          | Tenryubito    |

### Other Mods & Plugins

1. [Duels Plugin Commands](https://github.com/Realizedd/Duels/wiki/Commands): Type **/duel** in game to learn about our in game PvP system.
2. Factions X Plugin Commands: Type **/f help** in game to learn about building crews in game.
3. [Playtime Rewards Plus Plugin](https://nullforums.net/resources/playtimerewards-the-1-playtime-rewards-plugin.9617/): Type **/playtime help** in game to get started on claiming rewards.
